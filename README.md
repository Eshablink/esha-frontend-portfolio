# Dabbara Esha Portfolio Website

## What this site is
This project is a responsive single-page personal portfolio website for **Dabbara Esha**, a CS Engineering Student and Java & Python Developer. It is built as a single `index.html` file with embedded CSS and JavaScript, so it can run without any build tools or external frameworks.

The site includes:
- A hero section with Dabbara Esha's name, professional tagline, and a call-to-action button.
- An About Me section introducing academic interests and goals.
- A Skills section with grouped badges for languages, tools, and concepts.
- A Projects section with four project cards and GitHub link placeholders.
- A Contact section with sample contact details and a call-to-action.

## How to open it locally
1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` in your file explorer, or open it in any modern web browser.

Optional local server method:
1. Open a terminal in the project folder.
2. Run `python3 -m http.server 8000`.
3. Visit `http://localhost:8000` in your browser.

## JavaScript features used
The page uses pure JavaScript for a few interactive enhancements:
- **Dark/light mode toggle** in the navigation bar, with the user preference stored in `localStorage`.
- **Smooth scrolling** for navigation links and the “View My Work” button.
- **Typing animation** on the hero tagline to create a dynamic introduction effect.
